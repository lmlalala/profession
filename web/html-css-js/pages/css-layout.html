1. 选择器优先级 面试必考

优先级计算规则（权重） !important > 内联样式(1000) > ID(100) > 类/伪类/属性(10) > 标签/伪元素(1) > 通配符(0)
/* 权重示例 */ #app .list li:hover /* 100 + 10 + 1 + 10 = 121 */ .nav > a.active /* 10 + 1 + 10 = 21 */ div + p /* 1 + 1 = 2 */ * /* 0 */ /* 同权重：后写的覆盖先写的 */ /* !important 最高，但会破坏样式级联，慎用 */

常用伪类与属性选择器

2. 盒模型 面试必考

box-sizing 的区别 content-box（默认）：width = 内容宽，总宽 = width + padding + border
border-box（推荐）：width = 内容 + padding + border，总宽 = width
/* 推荐全局设置 */ *, *::before, *::after { box-sizing: border-box; } /* content-box（默认）*/ .box { width: 200px; padding: 20px; border: 5px solid; /* 实际占宽 = 200+40+10 = 250px */ } /* border-box */ .box { box-sizing: border-box; width: 200px; padding: 20px; border: 5px solid; /* 实际占宽 = 200px（不变）*/ }
盒模型结构
margin（外边距）
border（边框）
padding（内边距）
content（内容区）

5. 定位（Position）面试高频

值说明参照物脱离文档流
static默认，正常文档流—❌
relative相对自身原位置偏移自身原位置❌（占位）
absolute绝对定位最近的非static祖先✅
fixed固定在视口视口（viewport）✅
sticky滚动到阈值时固定最近滚动容器❌（混合）
常见面试题：absolute 的包含块 absolute 元素会找最近的 position 不为 static 的祖先元素作为包含块。 所以父元素通常设 position: relative 来"包住"子元素的绝对定位。
