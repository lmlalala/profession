1. 语义化标签 面试高频

为什么要语义化？

核心价值 ① 利于 SEO（搜索引擎更好理解内容结构）   ② 提升可访问性（屏幕阅读器）   ③ 代码可读性强，便于维护
❌ 非语义化（div 地狱）
<div class="header">... <div class="nav">... <div class="main">... <div class="sidebar">... <div class="footer">...
✅ 语义化
<header>页头</header> <nav>导航</nav> <main> <article>文章</article> <aside>侧边栏</aside> </main> <footer>页脚</footer>
页面结构可视化
📌 <header> — 页头，包含 logo、标题
🧭 <nav> — 主导航链接
📄 <main> — 主体内容（每页唯一）
📰 <article> — 独立可复用内容（文章、帖子）
📦 <section> — 章节，有标题的内容块
📎 <aside> — 侧边栏、相关内容
🔻 <footer> — 页脚、版权信息

其他常用语义化标签

标签语义使用场景
<figure> / <figcaption>图文组合图片 + 说明文字
<time datetime="...">时间发布时间、日期
<mark>高亮文本搜索结果高亮
<details> / <summary>折叠内容FAQ、折叠面板
<dialog>对话框模态框（原生支持）
<progress>进度条任务进度
<meter>度量值磁盘用量、评分

7. 原生折叠与对话框

<details> 原生折叠

什么是 BFC？
BFC（Block Formatting Context）块级格式化上下文，是页面中一块独立的渲染区域，内部元素的布局不会影响外部。
什么是闭包？
闭包是函数与其词法环境的组合，使函数可以访问其外部作用域的变量，即使外部函数已经执行完毕。
