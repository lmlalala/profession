1. 选择器优先级 面试必考

优先级计算规则（权重） !important > 内联样式(1000) > ID(100) > 类/伪类/属性(10) > 标签/伪元素(1) > 通配符(0)
/* 权重示例 */ #app .list li:hover /* 100 + 10 + 1 + 10 = 121 */ .nav > a.active /* 10 + 1 + 10 = 21 */ div + p /* 1 + 1 = 2 */ * /* 0 */ /* 同权重：后写的覆盖先写的 */ /* !important 最高，但会破坏样式级联，慎用 */
选择器类型示例权重
内联样式style="color:red"1,0,0,0
ID 选择器#header0,1,0,0
类 / 伪类 / 属性.active :hover [type]0,0,1,0
标签 / 伪元素div ::before0,0,0,1
通配符 / 关系符* > + ~0,0,0,0

2. 盒模型 面试必考

box-sizing 的区别 content-box（默认）：width = 内容宽，总宽 = width + padding + border
border-box（推荐）：width = 内容 + padding + border，总宽 = width
content-box（默认）
width: 200px
padding: 20px
border: 5px
实际占宽: 250px
border-box（推荐）
width: 200px
padding: 20px
border: 5px
实际占宽: 200px
/* content-box（默认）*/ .content-box { box-sizing: content-box; width: 200px; padding: 20px; border: 5px solid #6c63ff; /* 实际占宽 = 200+40+10 = 250px */ } /* border-box（推荐）*/ .border-box { box-sizing: border-box; width: 200px; padding: 20px; border: 5px solid #00d4aa; /* 实际占宽 = 200px（不变）*/ } /* 通用 */ .box-wrap { margin-bottom: 0.5rem; } .label { font-size: 0.8rem; color: #8892b0; margin-bottom: 4px; } .box { background: rgba(108,99,255,0.1); font-size: 0.8rem; line-height: 1.6; border-radius: 4px; } strong { color: #ffd166; }

3. Flexbox 弹性布局 面试必考

核心概念 父元素设 display: flex，子元素自动成为 flex item。主轴（main axis）默认水平，交叉轴（cross axis）垂直。

.flex-container { display: flex; flex-direction: row; /* 主轴方向：row | column */ justify-content: flex-start;/* 主轴对齐：flex-start | center | space-between */ align-items: stretch; /* 交叉轴对齐：stretch | center | flex-end */ flex-wrap: nowrap; /* 换行：nowrap | wrap */ gap: 0.5rem; min-height: 100px; padding: 0.5rem; background: #22263a; border-radius: 6px; } .flex-item { background: #6c63ff; color: #fff; padding: 0.5rem 0.75rem; border-radius: 4px; font-size: 0.85rem; font-weight: 600; display: flex; align-items: center; justify-content: center; min-width: 40px; } .flex-item:nth-child(2) { background: #00d4aa; color: #0f1117; } .flex-item:nth-child(3) { background: #ff6584; } .flex-item:nth-child(4) { background: #ffd166; color: #0f1117; } .tall { height: 60px; } .short { height: 40px; }
父容器属性值说明
flex-directionrow | column | row-reverse | column-reverse主轴方向
justify-contentflex-start | center | space-between | space-around | space-evenly主轴对齐
align-itemsstretch | flex-start | flex-end | center | baseline交叉轴对齐
flex-wrapnowrap | wrap | wrap-reverse换行
gap长度值子项间距
/* 子项属性 */ .item { flex: 1; /* flex-grow:1 flex-shrink:1 flex-basis:0 */ flex-grow: 1; /* 剩余空间分配比例 */ flex-shrink: 1; /* 空间不足时收缩比例 */ flex-basis: 200px;/* 初始大小 */ align-self: center; /* 单独设置交叉轴对齐 */ order: 2; /* 排列顺序（默认0）*/ }

5. 定位（Position）面试高频

值说明参照物脱离文档流
static默认，正常文档流—❌
relative相对自身原位置偏移自身原位置❌（占位）
absolute绝对定位最近的非static祖先✅
fixed固定在视口视口（viewport）✅
sticky滚动到阈值时固定最近滚动容器❌（混合）
常见面试题：absolute 的包含块 absolute 元素会找最近的 position 不为 static 的祖先元素作为包含块。 所以父元素通常设 position: relative 来"包住"子元素的绝对定位。
