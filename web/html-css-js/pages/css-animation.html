1. transition 过渡

语法：transition: [property] [duration] [timing-function] [delay]
timing-function 常用值：ease | ease-in | ease-out | linear | cubic-bezier(x1,y1,x2,y2)
all
color
cubic
delay

hover 查看过渡效果

.demo-row { display: flex; gap: 1.5rem; flex-wrap: wrap; align-items: center; padding: 1.5rem; } .box { width: 80px; height: 80px; background: #6c63ff; border-radius: 8px; display: flex; align-items: center; justify-content: center; color: #fff; font-size: 0.8rem; font-weight: 600; cursor: pointer; } /* transition: all — 所有属性都过渡 */ .trans-all { transition: all 0.4s ease; } .trans-all:hover { transform: scale(1.3) rotate(15deg); background: #00d4aa; border-radius: 50%; } /* transition: 指定属性 */ .trans-color { background: #ff6584; transition: background-color 0.3s, color 0.3s; } .trans-color:hover { background: #ffd166; color: #0f1117; } /* cubic-bezier 弹性效果 */ .trans-cubic { background: #00d4aa; color: #0f1117; transition: transform 0.5s cubic-bezier(0.34, 1.56, 0.64, 1); } .trans-cubic:hover { transform: translateX(60px); } /* delay 延迟 */ .trans-delay { background: #ffd166; color: #0f1117; transition: transform 0.4s ease 0.2s; /* 0.2s 延迟 */ } .trans-delay:hover { transform: translateY(-20px); } .tip { padding: 0 1.5rem 1rem; font-size: 0.8rem; color: #8892b0; }

2. transform 变换

性能提示 优先使用 transform 和 opacity 做动画，它们只触发合成（Composite），不触发重排（Reflow）或重绘（Repaint），性能最佳。

scale
rotate
skew
3D Y轴

.tf-grid { display: grid; grid-template-columns: repeat(4, 80px); gap: 1rem; padding: 1.5rem; } .tf-box { width: 70px; height: 70px; background: #22263a; border: 1px solid #2e3350; border-radius: 6px; display: flex; align-items: center; justify-content: center; font-size: 0.72rem; color: #8892b0; cursor: pointer; transition: transform 0.4s cubic-bezier(0.34, 1.56, 0.64, 1); } /* 2D 变换 */ .tf-translate:hover { transform: translate(10px, -10px); } .tf-scale:hover { transform: scale(1.4); } .tf-rotate:hover { transform: rotate(45deg); } .tf-skew:hover { transform: skew(15deg, 5deg); } .tf-matrix:hover { transform: matrix(1.2, 0.2, -0.2, 1.2, 5, -5); } /* 3D 变换：父元素需设 perspective */ .tf-3d { perspective: 400px; display: flex; align-items: center; justify-content: center; } .tf-3d-box { width: 70px; height: 70px; background: linear-gradient(135deg, #6c63ff, #00d4aa); border-radius: 6px; display: flex; align-items: center; justify-content: center; font-size: 0.7rem; color: #fff; cursor: pointer; transition: transform 0.5s ease; } .tf-3d-box:hover { transform: rotateY(180deg); } .tip { padding: 0 1.5rem 1rem; font-size: 0.8rem; color: #8892b0; }

/* ── 定义关键帧 ── */ @keyframes bounce { 0%, 100% { transform: translateY(0); animation-timing-function: ease-in; } 50% { transform: translateY(-40px); animation-timing-function: ease-out; } } @keyframes spin { to { transform: rotate(360deg); } } @keyframes pulse { 0%, 100% { transform: scale(1); opacity: 1; } 50% { transform: scale(1.15); opacity: 0.7; } } @keyframes shake { 0%, 100% { transform: translateX(0); } 20% { transform: translateX(-8px); } 40% { transform: translateX(8px); } 60% { transform: translateX(-5px); } 80% { transform: translateX(5px); } } @keyframes fadeInUp { from { opacity: 0; transform: translateY(20px); } to { opacity: 1; transform: translateY(0); } } @keyframes gradientShift { 0% { background-position: 0% 50%; } 50% { background-position: 100% 50%; } 100% { background-position: 0% 50%; } } @keyframes typewriter { from { width: 0; } to { width: 100%; } } @keyframes blink { 50% { border-color: transparent; } } /* ── 使用动画 ── */ .anim-row { display: flex; gap: 1.5rem; flex-wrap: wrap; align-items: flex-end; min-height: 120px; padding: 1rem 1.5rem; } .anim-box { width: 80px; height: 80px; border-radius: 8px; display: flex; align-items: center; justify-content: center; font-size: 0.7rem; font-weight: 600; color: #fff; } /* animation: name duration timing-function delay iteration-count */ .anim-bounce { background: #6c63ff; animation: bounce 1.2s ease-in-out infinite; } .anim-spin { background: #00d4aa; color: #0f1117; animation: spin 1s linear infinite; } .anim-pulse { background: #ff6584; animation: pulse 1.5s ease-in-out infinite; } .anim-shake { background: #ffd166; color: #0f1117; animation: shake 0.5s ease infinite; } .anim-fadein { background: #61afef; color: #0f1117; animation: fadeInUp 1s ease both; } /* 渐变流动背景 */ .anim-gradient { margin: 0 1.5rem 1rem; height: 60px; border-radius: 8px; background: linear-gradient(270deg, #6c63ff, #00d4aa, #ff6584, #ffd166); background-size: 400% 400%; animation: gradientShift 4s ease infinite; display: flex; align-items: center; justify-content: center; color: #fff; font-weight: 700; } /* 打字机效果 */ .anim-typewriter { margin: 0 1.5rem 1.5rem; font-family: monospace; font-size: 1rem; color: #00d4aa; overflow: hidden; white-space: nowrap; border-right: 2px solid #00d4aa; width: 0; /* animation-fill-mode: forwards — 结束后保持最终状态 */ animation: typewriter 2s steps(20) forwards, blink 0.75s step-end infinite; }
