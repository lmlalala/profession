# Vue 2 学习项目

基于 Vue CLI 的 Vue 2 学习项目，掌握 Options API 核心概念。

## 技术栈

- **Vue** 2.6 — Options API
- **Vue CLI** 5 — 项目脚手架与构建
- **ESLint** — 代码规范

## 开发

```bash
pnpm install

# 启动开发服务器
pnpm serve        # http://localhost:8080

# 生产构建
pnpm build

# 代码检查
pnpm lint
```

## 学习内容

- 模板语法与指令（v-bind、v-on、v-if、v-for、v-model）
- 组件通信（props、$emit、provide/inject、EventBus）
- 计算属性与侦听器（computed、watch）
- 生命周期钩子
- 插槽（slot、作用域插槽）
- 混入（mixin）
- 自定义指令
- 过渡与动画
- Vue Router 路由
- Vuex 状态管理
